## Gameplay

The game is a mix of an old Warcraft 3 mod called
[Rabbits vs Sheep](https://www.hiveworkshop.com/threads/rabbits-vs-sheep-1-0-62.80502/) and [Vampire Survivors](https://store.steampowered.com/app/1794680/Vampire_Survivors/).
So a PvP survivor-like.

Basically, you move around while your character automatically attacks based on the weapons you have. All players start with the same weapon, getting new weapons and upgrades on level up.

Once a player loses all their HP or has more than a certain amount of enemies (tbd.) on their side, they lose the game.

For now the game only supports split screen, which should work fine for playing locally with friends. Maybe I'll invest into creating proper network code one day, but for now I just wanna focus on getting the game ready.
